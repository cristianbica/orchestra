# AI Docs + Agents Template

Copy these into your repo root:

1) Copy `.ai/`
- Copy this repo’s `.ai/` folder into your target repository root.

2) Copy `AGENTS.md`
- Copy this repo’s `AGENTS.md` into your target repository root.

Optional: install tool-specific wrappers/templates

- GitHub Copilot (VS Code)
  - Copy `.ai/templates/github/copilot-instructions.md` → `.github/copilot-instructions.md`
  - Copy `.ai/templates/github/agents/` → `.github/agents/`

- Claude Code
  - Copy `.ai/templates/claude/CLAUDE.md` → `.claude/CLAUDE.md`
  - Copy `.ai/templates/claude/agents/` → `.claude/agents/`

- OpenAI Codex
  - Copy `.ai/templates/codex/AGENTS.md` → `AGENTS.md` (merge with your existing `AGENTS.md` if needed)

- OpenCode
  - Copy `.ai/templates/opencode/` → `.opencode/`
  - Move (and merge if needed) `.opencode/AGENTS.md` → `AGENTS.md`

How to use this system: see [.ai/HUMANS.md](.ai/HUMANS.md).
